## 0.0.1

- Added typed runtime API for SharedPreferences and flutter_secure_storage.
- Added PreferenceKey<T>, PreferencesStorageRouter and PreferencesService.
- Added reactive watch support for preference updates.
- Added built-in serializers for common complex value types.
- Added source_gen and build_runner based code generation with @Prefs and @Pref.
- Added example app, generated demo preferences and package documentation.
